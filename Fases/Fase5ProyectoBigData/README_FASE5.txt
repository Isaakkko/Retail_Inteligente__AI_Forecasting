En este entregable se intentó simular un proceso de calculo en tiempo real
insertando un dato manualmente y actualizando en tiempo real la base de datos y haciendo
una consulta para ver si el dato se insertó correctamente, para entregar y dar conocimiento de
la fase 5 de cálculos en tiempo real o streaming lo quisimos hacer con un solo dato para la
practica... Ya que para nuestro tema de proyecto lo vimos innecesario porque trabajamos con un
dataset de kaggle.
